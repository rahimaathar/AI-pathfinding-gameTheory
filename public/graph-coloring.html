Algorithm Comparison
Arc Consistency
  • Preprocessing step that removes inconsistent values
  • Reduces search space before backtracking
  • May not find a complete solution
  • Fast execution time
DFS Backtracking
  • Systematic search through all possible assignments
  • Guarantees finding a solution if one exists
  • Can be slow for large problems
  • Complete and sound algorithm
